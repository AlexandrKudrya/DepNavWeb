# src/

Исходный код приложения.

## Файлы

| Файл | Описание |
|------|----------|
| `main.ts` | Точка входа — монтирует App в DOM |
| `App.svelte` | Корневой компонент — загрузка данных, layout |
| `vite-env.d.ts` | TypeScript декларации для Vite |

## Папки

| Папка | Описание |
|-------|----------|
| `lib/` | Переиспользуемые компоненты, stores, утилиты |
| `styles/` | Глобальные CSS стили и тема |

## App.svelte

Главный компонент приложения:

```
┌─────────────────────────────────────┐
│  SearchBar (поиск)                  │
├─────────────────────────────────────┤
│                          FloorSwitch│
│                          ┌────┐     │
│      LeafletMap          │ ▲  │     │
│      (карта)             │ 2  │     │
│                          │ ▼  │     │
│                          └────┘     │
├─────────────────────────────────────┤
│  MarkerPopup (инфо о маркере)       │
└─────────────────────────────────────┘
```

**Состояния:**
- `loading` — загрузка JSON данных
- `error` — ошибка загрузки
- `ready` — карта готова к использованию
