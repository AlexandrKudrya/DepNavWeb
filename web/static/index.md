# Static

Статические файлы, копируются в build без обработки.

## Структура

| Папка/Файл | Назначение |
|------------|------------|
| `tiles/` | Тайлы карт (PNG) |
| `icons/` | SVG иконки маркеров |
| `favicon.svg` | Иконка сайта |

## Доступ

Файлы доступны от корня:
- `static/tiles/...` → `/tiles/...`
- `static/icons/room.svg` → `/icons/room.svg`
