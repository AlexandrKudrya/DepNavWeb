# Backend Source

Исходный код API сервера.

## Вложенные папки

| Папка | Назначение |
|-------|------------|
| `routes/` | Определение роутов и эндпоинтов |
| `controllers/` | Обработка HTTP запросов |
| `services/` | Бизнес-логика (работа с данными) |
| `models/` | Типы и схемы данных |
| `db/` | Подключение к БД, миграции, seed |
| `middleware/` | Express/Fastify middleware |

## Основные файлы

| Файл | Назначение |
|------|------------|
| `index.ts` | Точка входа, запуск сервера |
| `app.ts` | Конфигурация приложения |
| `config.ts` | Настройки (порт, БД, etc.) |
