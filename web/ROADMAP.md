# DepNav Web — Roadmap

## Версии

### v1.0 — MVP (текущая)

- [x] Интерактивная карта с зумом и перетаскиванием
- [x] Переключение этажей
- [x] Поиск по аудиториям
- [x] Информация о маркере при клике
- [x] Типы маркеров (комнаты, туалеты, лифты, лестницы)
- [x] Мобильный адаптивный UI
- [x] Светлая/тёмная тема (авто)
- [x] GitHub Pages деплой

---

### v1.1 — Интеграция с расписанием

**Цель:** Показывать где проходят занятия прямо на карте.

- [ ] Интеграция с [timetable.spbu.ru API](research/timetable-api.md)
- [ ] Выбор группы / преподавателя
- [ ] Отображение расписания на день/неделю
- [ ] Кнопка "Показать на карте" для занятия
- [ ] Подсветка аудитории текущего/следующего занятия
- [ ] Кэширование расписания в localStorage

**Исследование:** [research/timetable-api.md](research/timetable-api.md)

---

### v1.2 — PWA и офлайн

**Цель:** Работа без интернета внутри здания.

- [ ] Service Worker для кэширования
- [ ] Manifest.json для установки
- [ ] Офлайн-режим (карта + последнее расписание)
- [ ] "Установить приложение" баннер
- [ ] Push-уведомления о занятиях (опционально)

---

### v1.3 — Улучшения UX

- [ ] Анимации переходов между этажами
- [ ] Zoom к маркеру при выборе из поиска
- [ ] История поиска
- [ ] Избранные аудитории
- [ ] Шаринг ссылки на аудиторию (`?room=3343`)
- [ ] QR-код для быстрого доступа

---

### v2.0 — Навигация

**Цель:** Построение маршрутов внутри здания.

- [ ] Граф путей между точками
- [ ] Алгоритм поиска пути (A*)
- [ ] Отображение маршрута на карте
- [ ] Учёт лестниц и лифтов
- [ ] Голосовые подсказки (опционально)

**Требует:** Создание графа навигации для каждого этажа.

---

### v2.1 — Геолокация (экспериментально)

**Цель:** Показать текущее положение пользователя.

- [ ] Исследование indoor positioning
- [ ] WiFi fingerprinting (если доступно)
- [ ] Bluetooth beacons (если установлены)
- [ ] Fallback: ручной выбор "Я здесь"

**Примечание:** GPS внутри здания не работает.

---

### v3.0 — Мультикампус

**Цель:** Поддержка нескольких зданий/факультетов.

- [ ] Физический факультет (spbu-pf)
- [ ] Другие факультеты СПбГУ
- [ ] Переключатель зданий
- [ ] Общий поиск по всем зданиям
- [ ] Расписание с учётом разных корпусов

---

## Идеи для обсуждения

### Социальные фичи
- [ ] "Где мои друзья" — шаринг локации
- [ ] Отметки "Здесь свободно" для аудиторий
- [ ] Отзывы об аудиториях

### Информационные
- [ ] Расписание работы столовой/буфета
- [ ] Контакты деканата, кафедр
- [ ] Объявления факультета
- [ ] Мероприятия на карте

### Accessibility
- [ ] Маршруты для маломобильных
- [ ] Отметки пандусов, лифтов
- [ ] Голосовой интерфейс

### Интеграции
- [ ] Telegram бот (@depnav_bot)
- [ ] Виджет для ЛК СПбГУ
- [ ] Apple/Google Wallet для расписания
