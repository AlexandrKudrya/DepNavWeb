# Assets

Статические ресурсы приложения.

## Планируемое содержимое

| Папка/Файл | Назначение |
|------------|------------|
| `icons/` | Иконки маркеров (SVG) |
| `images/` | Изображения (логотипы, placeholder'ы) |
| `fonts/` | Шрифты (если локальные) |

## Иконки маркеров

Для типов маркеров из Android-версии:
- `marker-room.svg` — комната
- `marker-entrance.svg` — вход
- `marker-stairs-up.svg` — лестница вверх
- `marker-stairs-down.svg` — лестница вниз
- `marker-elevator.svg` — лифт
- `marker-wc.svg` — туалет
- `marker-wc-man.svg` — мужской туалет
- `marker-wc-woman.svg` — женский туалет
