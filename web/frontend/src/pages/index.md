# Pages

Страницы приложения (привязаны к роутам).

## Планируемые страницы

| Страница | Роут | Описание |
|----------|------|----------|
| `HomePage` | `/` | Главная с выбором здания |
| `MapPage` | `/map/:buildingId` | Интерактивная карта здания |
| `SearchPage` | `/search` | Расширенный поиск |
| `AboutPage` | `/about` | О приложении |
| `NotFoundPage` | `*` | 404 страница |

## Структура страницы

```
PageName/
├── PageName.tsx      # Компонент страницы
├── PageName.css      # Стили страницы
└── index.ts          # Реэкспорт
```
