# Frontend Source

Исходный код веб-интерфейса.

## Вложенные папки

| Папка | Назначение |
|-------|------------|
| `components/` | Переиспользуемые UI компоненты |
| `pages/` | Компоненты страниц (привязаны к роутам) |
| `services/` | Сервисы для работы с API и данными |
| `hooks/` | Кастомные хуки / composables |
| `utils/` | Вспомогательные функции |
| `styles/` | CSS/SCSS файлы |
| `types/` | TypeScript определения типов |
