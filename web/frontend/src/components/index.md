# Components

Переиспользуемые UI компоненты.

## Планируемые компоненты

- `Map/` — компонент карты (обёртка над Leaflet/OpenLayers)
- `Marker/` — маркеры на карте (комнаты, лифты, туалеты)
- `SearchBar/` — поиск по комнатам и маркерам
- `SearchResults/` — список результатов поиска
- `FloorSelector/` — переключатель этажей
- `MarkerInfo/` — карточка с информацией о маркере
- `BuildingSelector/` — выбор здания/корпуса
- `Modal/` — базовый компонент модального окна
- `Button/` — кнопки разных типов
- `Spinner/` — индикатор загрузки

## Организация

Каждый компонент в своей папке:
```
ComponentName/
├── ComponentName.tsx    # Основной файл
├── ComponentName.css    # Стили (или .module.css)
├── index.ts             # Реэкспорт
└── ComponentName.test.ts # Тесты (опционально)
```
